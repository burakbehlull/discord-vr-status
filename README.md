# Discord VR Status

**Install packages**:
``` npm i ```

**Start the status**:
``` npm start ```

<br />

In the ` settings.json ` section, you only need to enter your own account token:
```json
{
    "TOKEN": "",
    "GATEWAY_URL": "wss://gateway.discord.gg/?v=9&encoding=json",
    "DEBUG": false
}
```
